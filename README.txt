Article Transmogrifier

-----------------------------------------------------------------------
About

The application was programmed as a tool to make reading new articles 
more interesting. The idea was originally based on this comic strip,

https://xkcd.com/1288/

-----------------------------------------------------------------------
Technical

This is a google chrome browser extension, and was programmed using 
Jquery. 

To learn more about Jquery visit: https://jquery.com/
To learn more about Chrome Browser Extensions visit: https://developer.chrome.com/extensions
